Local DB InfoTor + http-server (Локальная поисковая система InfoTor)

Для пользователей Windows предварительно нужно установить на компьютер Python версии 3.* 
с официального сайта https://www.python.org/downloads/

Запуск локального сервера через start.cmd. 
Остановка - Ctrl+C и подтвердить выход (Y) в окне консоли.

Можно настроить в Windows Диспетчер служб IIS - описано в файле 'IIS_Win10.docx'.

В каталог /DB поместить файлы 'torrents.db3' и 'content.db3', полученные отдельно.
При появлении очередного обновления просто замените эти файлы в каталоге /DB на новые. 

Можно выполнить обновление самостоятельно:
	для обновления дамп очередного бэкапа БД RuTracker.org в формате XML берется с 
		http://rutracker.org/forum/viewtopic.php?t=5290461
	Файл распаковать и поместить backup.*.xml в каталог /UPDATE.
	Запустить в нем 'replace_tag.py' - он подготовит файл для загрузки.
	Дождитесь завершения выполнения (~10-15 минут).

	Если есть новый справочник форумов forums.csv, то его поместите в каталог /DB.
	Запустите режим "Обновление", выбрав необходимые параметры.

	Время полной загрузки и обработки 12Гб порядка 2,5 часов. 
	Если загружать только часть категорий - то меньше.
